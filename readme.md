# DevOps Class Assignment - Node.js App

This is a sample Node.js application for the DevOps class assignment.

## Application Structure
- Simple Express. js server
- Requires environment variables for configuration
- Dockerized for container deployment

## Running Locally
```bash
npm install
npm start
```

## Environment Variables
The application requires the following environment variables:
- `DB_HOST`
- `DB_USER`
- `DB_PASSWORD`
- `API_KEY`

See details.md for complete assignment instructions.
